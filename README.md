# VERILOG 4-BIT ADDER

## Requirement

```shell
sudo apt install iverilog gtkwave 

```

## Compile

```shell
make compile

```

## run

```shell
make all

```

## view

```shell
make view
```

> [!NOTE]
> After you open the GTKWave you should see some thing like the following

[!Alt text](wave.png "4-bit Adder Wave")

## clean

```shell
make clean
```
